Website Unblocker

Disclaimer: This tool may not work for all websites. Some sites implement security measures (like X-Frame-Options: DENY or Content-Security-Policy) to prevent their content from being embedded within iframes. Please keep this in mind when using the tool.

How It Works

This project creates a Blob URL that contains an iframe, allowing users to attempt embedding websites that do not explicitly block this method. Hereâ€™s the process:

  • Enter a website URL into the input field.
  • A Blob is generated containing an HTML page with an <iframe> pointing to the provided URL.
  • A Blob URL is created and opened in a new tab, displaying the website inside an iframe.

Usage

To use this tool:

  • Enter a website URL in the input field (e.g., https://example.com or example.com).
  • Click the "Unblock" button to open the site inside a new Blob-based tab.

Limitations

  • Websites with X-Frame-Options: DENY cannot be embedded.
  • Sites with strict Content-Security-Policy rules may also block this method.
  • Some browsers, especially mobile ones, may have additional restrictions.
